#CCNA Final Project

This project was built after finishing the CCNA course to apply the main networking concepts in a real network design.

#Project Idea

The project represents a company with 9 departments.
Each department has 5 computers connected to a department switch.
Every department switch is then connected to a multilayer switch.
The multilayer switch is connected to a router so the network can reach devices outside the local network.

#Project Goals

Practice the CCNA topics learned during the course

Build a simple and organized company network

Connect all departments in one network

Allow communication inside the network

Enable access to external devices through the router
